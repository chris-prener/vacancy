The data used to develop our initial listof vacant properties was culled from a number of different sources. The first three were considered primary indicators - if one or more were true, the property was considered vacant in both the low and the high estimates.

Table 1: Data Sources
Indicator Source Description
(1) Vacant Lot (A) Property Ownership Database Represents the City’s assessment of the property as a vacant lot.
(2) Land Reutilization Authority (LRA) Property (A) Property Ownership Database and (B) LRA Sale Data Combines data on property ownership with the LRA’s own sales list to identify properties held in the City’s land bank.
(3) Land Clearence for Redevelopment Authority (LCRA) Property (A) Property Ownership Database Identi es properties held by the City’s development agency.
